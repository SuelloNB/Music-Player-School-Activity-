# Music Player (School Activity)

A simple web-based music player created as a school activity. This project allows users to play favorite songs, view album covers, and read lyrics in an interactive and user-friendly interface.

## Features

- Play and pause music
- Display song title and album cover
- Show lyrics for each song
- Organized music and image assets
- Built using HTML, CSS, and JavaScript only
